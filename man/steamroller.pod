=head1 NAME

steamroller - An AUR helper that creates a local repo from which pacman installs packages.

=head1 SYNOPSIS

B<steamroller> S<-S package name ...>

B<steamroller> S<-Su>

B<steamroller> S<-Ss search term>

B<steamroller> S<-U path/to/taurball.tar.gz>

=head1 DESCRIPTION

Steamroller is an AUR helper that places built packages into a local repo where L<pacman(8)> can install from.  Steamroller is designed to have similar syntax to pacman.

=head1 OPTIONS

=over 4

=item -S, --sync

Sychronise specified package list from AUR, resolving AUR dependencies, and add the built packages to the local repo.

=item -u, --update

When passed with \-S, check the AUR for updates to packages in the local repo and build 

=item -s, --search

When passed with \-S, perform a search of the AUR for the string provided.

=item -U, --upgrade

Build packages from specified taurballs and add them to the local repo.

=back

=head1 FILES

=over 4

=item F</etc/steamroller.conf>

The system-wide configuration file.

=item F<~/.steamroller.conf>

Per-user configuration file.

=back
